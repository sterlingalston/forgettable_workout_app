How to Create a GitHub Token

LIFT uses a free GitHub Personal Access Token to back up and sync your workouts across devices via a private GitHub Gist. No billing required.

  1. Go to github.com/settings/tokens/new (sign in if needed).
  2. Under Note, enter something like LIFT Workout App.
  3. Under Select scopes, check gist only — no other permissions are needed.
  4. Click Generate token at the bottom of the page.
  5. Copy the token (it starts with ghp_) — you won't be able to see it again.
  6. Come back here, enter your GitHub username and paste the token in the Account section above, then tap Sign In.

Your token is stored only on this device and is never shared. It is used exclusively to read and write your private gist.

Contribute Exercise Data

Help improve the app for everyone by contributing better videos, thumbnails, or instructions for any exercise. Each exercise is a single JSON file — no merge conflicts with other contributors.

  1. Fork the repo at github.com/sterlingalston/forgettable_workout_app.
  2. Create a file at data/community/<slug>.json where the slug is the exercise name lowercased with spaces and special characters replaced by hyphens.
    Example: Barbell Bench Press → barbell-bench-press.json
  3. Use this structure: 
    {
  "exerciseName": "Barbell Bench Press",
  "slug": "barbell-bench-press",
  "videoId": "rT7DgCr-3pg",
  "thumbnailUrl": null,
  "timed": false,
  "instructions": [
    "Lie flat on the bench under the bar.",
    "Grip just outside shoulder-width.",
    "Lower the bar under control to mid-chest.",
    "Press back to full lockout."
  ],
  "meta": {
    "contributor": "your-github-username",
    "source": "https://youtube.com/watch?v=rT7DgCr-3pg",
    "addedAt": "2026-03-18"
  }
}
    videoId is the YouTube ID only (e.g. rT7DgCr-3pg from youtube.com/watch?v=rT7DgCr-3pg). Shorts work too.
  4. Add your slug to the slugs array in data/community/index.json (keep it alphabetical).
  5. Open a Pull Request — that's it.

Community data takes priority over the automatic YouTube search. Videos you edit yourself via the ✏️ button always override everything.
